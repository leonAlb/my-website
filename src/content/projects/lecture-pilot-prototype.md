An AI-powered VS Code extension that helps university lecturers create structured lecture content — from outline to slides, teleprompter scripts, and quiz questions — using LLMs directly inside the editor. The extension provides a seamless workflow for educators to generate comprehensive lecture materials, enhancing the teaching experience and enabling efficient content creation with a predefined DSL-structure. The DSL is a Markdown-based extension using specific tags to provide a machine-readable structure to lecture materials, allowing AI systems to easily identify and generate content like scenes or quizzes.
